Command: mkpizza (DOS)

  MAKEPIZZA is a dummy command-line program that educates people how to use the
  Command-Line Interface (CLI).

  The DOS version is offered in three major variants:

  * MKPIZZA.EXE for DOS, CP/M, or legacy Windows Batch command-line program
    conventions
  * MKPIZZAU.EXE for general, UNIX/POSIX command-line program conventions
  * MKPIZZAP.EXE for PowerShell cmdlet conventions

Syntax:

  MKPIZZA [/N ] [/D ] [/P|/P:YES|/-P|/P:NO]
    [/S|/S:YES|/-S|/S:NO] [/E ] [contents]

Options:

  Basic Options:
  contents  Add some content to your pizza!
  /N=[name] Give your pizza some name! (optional).
  /P:[YES|NO] Add pineapple to the pizza (default: YES).
  /?        Display help.
  /V        Display version.
  Advanced Options:
  /D:[duration] Delays the execution of this program, which should have been
            done in less than 1 second (default: 0).
  /E:[exitcode] Change the exit code of this program, after the pizza is made.
            (default: 0)
  /S:[YES|NO] Ensure that only root, sudoer, or Administrators can make this
            pizza. Since DOS is a single-user operating system, setting this
            option to YES will decline your request (as a non-superuser).

Comments:

  Note that the names of the POSIX (MKPIZZAU) and PowerShell (MKPIZZAP)
  command versions for this DOS edition differ from their original names, which
  should be "makepizza" and "New-Pizza" respectively. This change is intended
  so the file names still fit within the DOS 8.3 file naming constraints.

  Since the DOS edition of MAKEPIZZA is written in C instead of Go, there may
  be little changes on how different-style command parameters are being parsed
  against the original software.

Examples:

  This command creates a new pizza named Johny, using the `/N` parameter.

  MKPIZZA /N=Johny "Johny, the Pizza"

  Apparently, this program has the algorithmic affinity to put pineapples on
  pizza. Oh, yuck! Make another one without it.

  MKPIZZA /N=Johny /P:NO "Johny, the Pizza"

  Nice! You'd think cooking the pizza longer could mean better, crispier piece
  of bread! Try to cook for 67 seconds while you put the fries in the bag.

  MKPIZZA /N=Johny /P:NO /D:67 "Johny, the Pizza"

  You'd think cooking a pizza for 67 seconds is insecure by default. Skibidi
  people could have tried to poison your pizza. Therefore, to maintain supply-
  chain security, only allow `root`, `sudoer`, or Administrator users to make
  the pizza. And just to be secure, please return an exit code of `165` instead
  of `0`.

  MKPIZZA /N=Johny /P:NO /D:67 /S:YES /E:165 "Johny, the Pizza"
